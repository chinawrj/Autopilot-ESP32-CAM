## 完成内容

### WebSocket 视频流架构

- **ws_stream.c** (170 行): WebSocket 流管理
  - FreeRTOS 流推送任务 (pinned to Core 1)
  - 最多 4 个并发 WS 客户端
  - Mutex 保护客户端列表，发送时不持锁
  - httpd close callback 自动清理断开的客户端
  - 独立 FPS 统计

- **stream_udp.html** (135 行): WebSocket 流前端
  - Canvas 渲染 (Blob → Image → drawImage)
  - 客户端 FPS 计算
  - 自动重连 (2s 延迟)
  - HUD 叠加: FPS + 温度 + WS 状态
  - LED toggle 按钮

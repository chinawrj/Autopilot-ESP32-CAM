## Work Done

### Browser Test Expansion (26 → 39 tests)
New tests added covering previously untested endpoints:
- **OTA status** — `GET /api/ota/status` returns idle state
- **SD status** — `GET /api/sd/status` returns mounted=true
- **SD list** — `GET /api/sd/list` returns file listing
- **MJPEG stream** — Port 8081 returns `multipart/x-mixed-replace` content-type
- **Camera POST** — Set quality to 20 and verify response
- **LED on/off cycle** — Full on→verify→off→verify cycle
- **LED bad JSON** — Malformed request returns 400
- **404 handling** — Unknown route returns 404
- **Status details** — Version, WiFi, RSSI fields present
- **CORS header** — `Access-Control-Allow-Origin: *` on snapshot

Key debugging:
- MJPEG stream test requires navigating browser away from homepage first (releases active MJPEG connection so ESP32 can serve curl)
- Persistent browser context corruption after improper shutdown — fixed by cleaning `~/.patchright-userdata`

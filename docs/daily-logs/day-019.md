## Issues Discovered

### Issue 1: Port 81 Intercepted by Network Transparent Proxy (Critical)

**Symptoms**:
- `curl http://192.168.1.171:81/stream/tcp` returns `Server: nginx`, 404

**Root cause**:
- Router/network has an nginx transparent proxy intercepting port 81 traffic
- `192.168.1.171:81` and `192.168.1.1:81` return the same "Index of /" page
- ESP32 serial logs confirm stream server did successfully start on port 81

**Fix**:
- Changed MJPEG stream server port from 81 to 8081
- Updated stream URL in index.html
- Updated port references in all tool scripts and documentation

**Files changed**:
- `main/stream_server.c` — port 81 → 8081
- `main/stream_server.h` — comment updated
- `main/index.html` — stream URL port updated
- `main/main.c` — comment updated
- `tools/quality_audit.py` — new quality audit script

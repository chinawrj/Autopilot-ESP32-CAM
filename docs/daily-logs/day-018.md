### OTA 端到端测试

**问题发现**: OTA 软重启后 OV2640 I2C 探测超时 (17 次 I2C timeout → "camera not supported")

**修复过程** (3 次迭代):
1. 延长 PWDN 时序 100ms→500ms + I2C pin reset → 失败
2. 添加 I2C bus recovery (时钟脉冲 bit-banging) → 失败
3. **`esp_camera_deinit()` before `esp_restart()`** → ✅ 成功

**根因**: 软重启不会重置 I2C 外设状态，必须在重启前正确释放

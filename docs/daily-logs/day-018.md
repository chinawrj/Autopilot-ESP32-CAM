### OTA End-to-End Test

**Issue discovered**: OV2640 I2C probe timeout after OTA soft reboot (17 I2C timeouts → "camera not supported")

**Fix process** (3 iterations):
1. Extended PWDN timing 100ms→500ms + I2C pin reset → Failed
2. Added I2C bus recovery (clock pulse bit-banging) → Failed
3. **`esp_camera_deinit()` before `esp_restart()`** → ✅ Success

**Root cause**: Soft reboot does not reset I2C peripheral state; must properly release before restart

### Technical Approach
- **Mode**: 1-bit SDMMC (only needs GPIO2/14/15, avoids GPIO4 LED conflict and GPIO12 MTDI issue)
- **Mount Point**: `/sdcard` (VFS FAT)
- **API**: `GET /api/sd/status`, `GET /api/sd/list`, `POST /api/sd/capture`, `GET /api/sd/file/*`, `POST /api/sd/delete`
- **Frontend**: New collapsible "SD Card Storage" panel on homepage

### Risks and Dependencies
- SD card physical insertion state unknown — Need graceful handling when no card inserted ✅ Implemented
- GPIO12 has MTDI conflict in 4-bit mode — Using 1-bit mode to avoid ✅
- GPIO4 (Flash LED) shared with SD DATA1 — Using 1-bit mode to avoid ✅

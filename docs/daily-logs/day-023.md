### 技术方案
- **模式**: 1-bit SDMMC (仅需 GPIO2/14/15，避免 GPIO4 LED 冲突和 GPIO12 MTDI 问题)
- **挂载点**: `/sdcard` (VFS FAT)
- **API**: `GET /api/sd/status`, `GET /api/sd/list`, `POST /api/sd/capture`, `GET /api/sd/file/*`, `POST /api/sd/delete`
- **前端**: 主页新增可折叠 "SD Card Storage" 面板

### 风险与依赖
- SD 卡物理插入状态未知 — 需要优雅处理未插卡情况 ✅ 已实现
- GPIO12 在 4-bit 模式下有 MTDI 冲突 — 使用 1-bit 模式规避 ✅
- GPIO4 (Flash LED) 与 SD DATA1 共用 — 使用 1-bit 模式规避 ✅
